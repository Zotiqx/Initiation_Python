À propos du projet

Ce générateur de motifs géométriques a été réalisé dans le cadre d’un projet universitaire Python (5 ECTS). Il utilise Flask pour l’interface web, Turtle pour les dessins, et Pillow pour convertir les images.

Réalisé avec ❤️ par un étudiant passionné de code, de géométrie et de design.
