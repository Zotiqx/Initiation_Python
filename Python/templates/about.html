> À propos

PolyDraw est une application web qui permet de générer dynamiquement des motifs géométriques via une interface rétro stylisée.

:: Fonctionnement

L'app utilise Turtle pour dessiner un motif en boucle, avec rotation à chaque tour, selon les paramètres entrés.

:: Outils

  • Python - logique principale
  • Flask - serveur web
  • Turtle - dessin
  • Pillow - conversion PNG
  • HTML/CSS - rendu visuel

:: Paramètres

  • nb_cotes : côtés du polygone
  • profondeur : répétitions
  • taille : longueur des lignes
  • angle : rotation entre chaque itération
  • couleur : en français ou hexadécimal
